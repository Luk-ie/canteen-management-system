Canteen Management System

- A comprehensive digital solution for school canteen management with sales pattern analysis and food waste reduction.

Features

- Dashboard : Real-time KPIs and sales trends
- Data Entry : Easy sales and waste recording
- Analytics : Sales patterns and menu performance
- Forecasting : Demand prediction for meal planning
- Waste Analysis : Food waste tracking and reduction
- Inventory : Stock level monitoring
- Export : Excel integration for manual analysis
